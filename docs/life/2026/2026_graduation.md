26.6.4

&nbsp; &nbsp; &nbsp; &nbsp; 研究生三年时间可真快，这就要毕业了。总想写点东西，感觉脑子思绪万千，又一时间不知从何写起...就最后安静坐在图书馆这个对我最熟悉的地方随意写写吧，或许想到哪就写到哪吧，算是对研究生这三年总结记录一下。

    读研这三年不后悔！本身读研算是本科一直以来的执念吧，一方面因为学校太差一直想向上走去更好的学校看看（PS. ~~电专太破了，也啥没可看的...~~），另一方面其实当时是有点希望未来有机会读博的hhhh（~~读不了一点...~~）
    
    当时下决定二战也不后悔，如果当时二战没考上不知道如今的自己是何样子?或者当时一战结束去了春招找的一般工作会是怎么样? 一切都不得而知了...无法预料没多久AI就发展如此迅速，之前还是一行一行写代码，没几年代码都用AI生成了，谁还用传统方法，变化太快了。斗胆猜想一下大概率我在小公司肯定也早被裁或因为压力大主动离开了...
    
    后面侥幸二战考上了(当年分真没很高，很是幸运)，读研之后也彻底打消了读博的想法，只求能顺利毕业就行！作为关门弟子硕士期间有折磨，但对比系里面别的狠老师，自身这折磨基本可以忽略，甚至可以说是幸福了。

    **关于收获**：在科研上似乎感觉并没太大收获，譬如收获什么好成果什么好科研idea，坦白讲到最后对于论文仍还是挺抗拒的，可能收获也就是即便面对这些"折磨"也能静下来好好看文献、写文献，最终也算独自完成了小论文+四五十页/四万多字的论文，尤其是这个密码学相关的领域，光看论文就很是头大。对于本科期间的自己是不敢想象了（翻了一下当时毕业论文1w多字，参加文献刚10几个...🤣）不管了反正不厉害也得厉害hhhhh，现在想想似乎当时也并没很难，一切熬过去就好了。更多的收获或许是读研过程中生活上的经历和感悟...

&nbsp; &nbsp; &nbsp; &nbsp; 回想起最开始联系导师，由于可能联系的晚点，也没最终选上理想的导师，开始一志愿选的导师是[张老师](https://cs.ncepu.edu.cn/szzk/zgml/jsjkxyjsjys/jsjzr/7f4f046545714c23aadfb8f6b2ffa2cb.htm)，二志愿[熊老师](https://cs.ncepu.edu.cn/szzk/yjsds/znjsyjqsjyjz/7164a14b2e864d2dabafd04b25b49a9d.htm)。只联系了一志愿，二志愿根本就没联系...最后分配调剂到了另一个[张老师](https://baike.baidu.com/item/%E5%BC%A0%E5%B0%91%E6%95%8F/42704)手下，虽然分配的张老师不算很差，但开始选的那个张老师似乎更好hhh（果真开始第一眼就没看错），坦白讲最开始选导师很避雷夫妻档，万万没想到最后还是分配到这边来了（命呀!）...好在没分配到sxq手下，这样一想这不幸运多了🙏 23.8.18偶尔间刷微博认识了同华电的校友『Yanni』，第一印象就是非常漂亮hhh，算是第一个认识的同学了，后面还是前后相继挨着入群的hhh。正式入学后，分到了宿舍楼——单身楼305(这楼名预兆就不好...)。开始宿舍帅是提前到了，健哥还没到，23.9.7那天我到之后帅出去了并没在宿舍住，也就没见到他。那天把各种东西收拾好之后，印象中认识了方哲，还和他骑车去买床垫子之类的...也是个挺坦率的小伙子，后面一个个都陆续认识的，尤其印象中有阵子貌似是佳昌闲的没事(他宿舍都很内向，除他之外)第一次来我们宿舍串，因为宿舍三人间，我在最外面靠门位置，刚进来第一句就对我说：“同学，你好帅...”，给整笑了🤣hhhh...（宿舍一直印象深刻，怎么其余俩舍友不帅了吗hhh）。虽说没多久班里面同学就基本都能认识了，但有的说话并不多，当时也算是班里面人认识最快的了（其实自身也没意识到，不知不觉怎么都认识了...）。坦白讲读研后和大学变化还是挺大的，就大学没认识几个人，赶上换专业又赶上疫情，似乎本科印象并没太多，尤其毕业而研究生不知不觉认识了好多，怎么样我都不后悔这段读研期间有折磨但更美好的经历！

&nbsp; &nbsp; &nbsp; &nbsp; 这三年主要经历来讲：**研一上课+玩，研二小论文+玩，研三找工作+大论文+毕业+玩**，最终主要目标是能顺利毕业+找到尽可能满意的工作。==**研一**==: 说起来上课，本科期间印象中是一节课都没逃过（~~太老实了~~），读研期间大部分也都去上课了，偶尔个别逃过课hhh，坦白讲这课程真用处并不大，对论文也没啥帮助。当时选课，完全是导师决定的，印象中上学期课程相对还多点，下学期课程更少了，但不论咋样起码算是学生时代在教室里的宝贵回忆了，以后再想坐教室里哪还有机会！现在想想研一这一年可以说真非常舒适，上课也没啥太大压力，基本的日常就是上课、吃喝、图书馆值班、刷剧、玩耍。下学期开启了健身锻炼，没人疼结果练的浑身疼了...对于科研基本上也可以说是一点没整，一学年两个学期都是学期结束开个组会，也都是导师介绍介绍科研论文相关的。**==研二==**：研二上学期感觉就是看文献、整开题，写开题报告文献综述什么的，天呐，印象中当时差不多就每两周进行开组会汇报了，而且都是去二校那边教室里，一个个上台讲，坦白说感觉也没啥大用。后续开始弄小论文，一个小论文弄了好久好久...25.3.9就写好了发给老师，直接批了一顿（因为用的是电力科学与工程的模版...），之后改了八九版又翻译英文陆陆续续整到了七月多（~~有时候感觉导师给的意见也有点扯淡~~），投英文了几次，SCI二三区的，基本上没多久就被拒了，当时就准备让投学校校刊了，结果我当时碰到校刊编辑，聊了聊说开会把我们这个方向的给取消了，以后再不收了（垃圾方向!），当时觉得保底不了了，也不知道投啥了(师门之前就是除了英文期刊就是中文校刊)，后续得知有个同学投了个[网络信息安全](http://netinfo-security.org/CN/1671-1122/home.shtml)，期刊挺好的，公安部主管的。七月多投的，到了正式开学九月多仍没有消息，再之后石沉大海...，当时同门投的[电力信息与通信技术](http://ntps.epri.sgcc.com.cn/dlxxtx/CN/home)非常快，不到一个月就返修了，不到两个月差不多就发录用通知了（就是见刊慢点）。当时中文石沉大海，后面老师9月中旬收到了一个期刊的邀稿通知——[Cryptography](https://www.mdpi.com/journal/cryptography)，就在原有基础上赶快修改，弄了篇英文，9.20左右投了过去差不多，差不多一个月给返修了，后续11.11录用了，但但但最后毕业前提交资料才发现这篇原来是ESCI，而不是SCI，老师也没了解清楚就投了，甚至看的什么百度经验写的是SCI（太坑太坑了...），不过也怪我没了解清楚，但想着导师投的咋样得符合条件呢，不过咋样起码是有论文了，后续又挂了sxq项目，好在能满足毕业条件了，可以说研二这基本都在整论文了，研二暑假投出去论文后学了一阵子基础行测，但感觉也没帮助多少，仍好多不会...**==研三==**：到了研三，就开始各种投递简历了，各大网站投的都烦了，而且填各种信息挺麻烦，而且没在各大APP上海投，而是各官网上，^^各种投 + 改论文 + 复习复习行测^^，后续来回参加笔试，面试没收到几个，跑了不少城市——济南/石家庄/天津/北京/青岛（这还没投太远的，不然跑得更多...）最后秋招以家那边ICBC结束了，本来犹豫签不签的，还是保底签了...再研三下学期就完全写大论文了，也基本是刚开始写，大体思路就是模仿师兄的 + AI改，老师也提供不了太大帮助，后面春招基本也没投了。文职笔试有优势，3月多进面后正常发挥，也算录用了最后都准备去文职了（但公示巨慢，想着后续再投投别的），没想到公务员补录进面，想着试试吧，三月多面完最后成绩发现竟然公务员逆袭了（很是幸运），再之后更春招没投过了...就完全是整毕业论文了，最后五月中旬顺利完成答辩，答辩完一直等到五月底才公示了，我想大概率要去了，虽然会分到县城，也不清楚会怎么样...**一切都是未知**

    &nbsp; &nbsp; &nbsp; &nbsp; 25.6.5开始投了投 ==**暑假实习**==，陆续差不多投到了七月中，所有的全是石沉大海，就有个江苏电信的笔试，当时线上还是单机位，还完全自己硬做的，不出意料笔试完就没消息了。自身投的全是国央企这种——银行、运营商、研究所以及一些暑假开放日...没投什么私企，所以相对门槛高，机会也更少一些，没拿到什么机会确实也挺正常。再之后九月初 ==**秋招**== 就算正式开始了——银行、运营商以及管网、华电、国电投、中石油、中石化、山能等，有山东的岗位基本都投山东的了，**总共收到笔试得有50个，收到面试也就13个，算是OC的就3个——ICBC/公务员/军队文职**，笔试和面试不是所有的都参加了，^^笔试考察的内容主要侧重行测+专业课，面试有无领导、半结构化、包括公务员的单盲结构化，基本就是问简历和聊天，技术和项目有的会问，但都不会很深入^^。国央企笔试基本都是线上（石油石化就直接线下，还是当场出分的），并且双机位的多，事业单位大部分都线下，记得最早是有个林草局的事业单位（西安的大数据中心专业技术岗），算是第一个事业单位了，9月多就开始笔试了，北京考的行测(60分)+申论(40分)，学得不咋地，能进面也挺不容易，尤其这种纯行测申论，不带专业课的，更看这方面学得怎么样了。同样有次天津事业单位，是缴费那种，当时缴费了差不多300，招一个（不少事业单位其实不用缴费就能报名），带专业课那种，考察计算机基础知识，问答选择判断甚至编程填空啥都有，最后排名11，专业课83.5，我当时觉得做的并不好，要是计算机基础好些拿到这个分数问题不大，甚至85分以上也是有机会的，但行测我就50多，第一名行测70了，行测达到70水平没好好练过或者天赋不够真不容易达到，所以就是说在保持专业课的情况下，行测越好可能机会相对就会多一些，另一个方面说如果行测很好，完全考行测这种也机会会不小，就比如班里走考公的山东同学，行测很好，国考130多（进面了），省考140（报济南没进面），最后靠行测山东烟草也录用了，我也去考了，但就考了70完全是不够的。不过话又说回来行测这玩意有时候也挺玄学，可能学一阵子并没太明显的提高，我感觉我是这样，也可能学行测不得法，不过想想当时刷题确实也并不很多，不知不觉在边学边投边改论文过程中，秋招就结束了，也跑了不少地方，参加了也不少，但最后并没收获什么太好的offer，不过想想也算是正常。
    
    &nbsp; &nbsp; &nbsp; &nbsp; ==**关于校招**==：当时我基本算是放弃了，来学校招聘的基本是各大能源相关的（学校特色太明显），坦白讲主要还是招电气相关的，要说有没有对口招的，肯定是有，前提是不挑地方，就山东的基本没有啥，印象中有山东电工电气、泰开等，当时也没积极去二校宣讲会去看看，线上是投了投也没消息了。如果是不挑地方，完全看重公司，感觉计算机不少还是有机会的，走线下校招是个不错的选择。此外，感觉系里面大不多数都备考电网，就河北电网这种相对网申好些（也可能本身华电在河北优势大些），就考上河北很多，就我学硕班里进电网的得有一半20个了，可见大部分都在考电网，但前提得网申有机会过，不过我这种执着山东电网确实真有点不容易，也很玄学，不好说报哪个地区网申就过了，今年报的家附近县就没过，不过当时也没完全在复习电网，因为感觉山东也难过，但要是不执着于山东电网，考辽宁/安徽/湖北，甚至河北应该都有会机会的，听说系里面今年辽宁还有去大连这种挺不错城市的，此外，了解的湖北不卡三方，网申不很难，但就是得分数高才有机会，专注于网申好过的地方，剩下的好好复习考上电网机会也是不小的，当时春招时候二批网申过了，最后还是放弃去参加了，不过话又说回来网申能不能过都不好说...有的复习好久网申没过的也是常见的，谁不保证不了百分之百过不去，归根到底还是自身抉择了，毕竟也是可以三个志愿可以选的，有的单位难有的单位是相对简单的，甚至[国网信产](http://www.sgitg.sgcc.com.cn/)有的单位可以走学校提前批。
    
    &nbsp; &nbsp; &nbsp; &nbsp; ==**最后春招**==：肯定也是有机会的，但相对会少一些，就校招来学校的也少了。但会有不少事业单位开始招聘，有的是海笔，但有的并不是，几个给了笔试但都放弃了，比如山东中烟等，就不想再考了，于是也放弃了。再次投了农行山东省本部，果然笔试都没给...春招虽然投了点但和不投区别并不大，因为给笔试的都放弃了

    &nbsp; &nbsp; &nbsp; &nbsp; **总得来说**，起码是找到工作了，相对还是不错，能接受。要说很满意嘛，也并没，理想可能是济南找个不错的，毕竟济南这城市还是真挺不错的，但其实真没啥机会，石沉大海。秋招拿到了家那边ICBC，想着咋样起码离家近，就几公里，也能接受，当签了之后就推送各方面负面消息，不过负面消息存在才是正常的。后面文职差不多没问题，觉得军队文职不错了（待遇来看确实是的），但又刷到各种问题——管理以及加班各方面都不好说，很开盲盒，大概率会很忙很累，我相信也确实存在，军队也是不好混，而且这没军籍没军衔，后续怎么着也不好说，要是没别的机会去的话也确实能去（不过文职相对确实比公务员容易多了，可能数学基础也算可以），当时如果选个家当地人武部或者某高校的话可能会比济南这个岗位更好些了，但也都不好说，因为报名也是根据人数报的，觉得报的少招的还挺多最后就报了。最后又公务员录用了，又开始纠结了，虽在家那边但毕竟不是家附近，算是家城市县城，而且又再次刷到各种劝退的。不过想想确实是啊，哪有什么完美的工作，离家近工作闲薪资高还稳定？纯纯做梦呢，要是有也轮不到我呀！话又说回来真济南不错工作了，难道就完美了，不会遇到各种问题了？显然肯定不是的呀！哪样的工作真干了以后都会有烦心事，会有各种问题呢，就完全要看自身能否适应，别人的感受并非自身感受呢，就根据目前了解到的做自己内心的判断就好了，选了就没必要后悔什么了！不过想想如果公务员工作规律，完全符合自身作息规律，这不工作已经是挺好了吧，你追求的不就如此嘛？当然就是没啥钱，富不了但也穷不了，但养活自己是完全够了。学生 --> 职场。未来还会很长很长...

**研究生 VS 本科**：
