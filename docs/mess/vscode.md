## 快捷操作

`ctrl + b` 打开和关闭侧边栏 --> 打开后按`ctrl + shift + e` 查看当前工作目录
  
- 前提： 键位不冲突

**`ctrl + shift + p` 开启命令选择区，和VScode进行聊天的窗口**

`ctrl + ~` 在编辑器和terminal进行来回切换

`ctrl + \` 左右分栏（~~抄代码~~）`ctrl + 数字` 可进行切换
